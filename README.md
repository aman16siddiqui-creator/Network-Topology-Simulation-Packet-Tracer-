# Network Topology Simulation using Cisco Packet Tracer & Wireshark

## Project Overview

This project focuses on designing and analyzing a computer network using **Cisco Packet Tracer** and **Wireshark**. A Star Topology network was simulated with PCs, a server, and networking devices to understand real-world network communication and protocols.

The project demonstrates network configuration, packet analysis, and troubleshooting using different networking concepts.

## Technologies Used

- Cisco Packet Tracer
- Wireshark
- Computer Networking Concepts

## Implemented Concepts

- Star Topology Design
- IP Addressing & Connectivity Testing
- ICMP Packet Analysis
- ARP & Ethernet Traffic Analysis
- SSL/TLS Communication Analysis
- VLAN Configuration
- DHCP Server Configuration
- OSPF Dynamic Routing
- UDP Traffic Analysis

## Features

- Designed a simulated network environment
- Configured devices with IP addresses
- Tested communication using ping
- Captured and analyzed network packets using Wireshark
- Implemented VLAN segmentation and DHCP services
- Configured dynamic routing using OSPF
